FORMAT EXACT POUR IMAGE GENERATION - SUIT CETTE STRUCTURE:
```python
from groq import Groq
import os
import base64

def generate_image(prompt, size="1024x1024", quality="standard", style=None):
    """
    Génère une image depuis une description textuelle.
    
    Args:
        prompt: Description de l'image à générer (str)
        size: Taille de l'image "WIDTHxHEIGHT" (str, défaut "1024x1024")
        quality: Qualité "standard" ou "hd" (str, défaut "standard")
        style: Style de l'image (optionnel, ex: "vivid", "natural")
    
    Returns:
        dict avec clés:
            - image_base64: Image encodée en base64
            - size: Taille de l'image
            - format: Format du fichier (ex: 'png')
            - prompt_used: Prompt utilisé
            - revised_prompt: Prompt amélioré par le modèle (si disponible)
    """
    # Step 1: Validate input
    if not isinstance(prompt, str) or len(prompt) < 5:
        raise ValueError("prompt must be at least 5 characters")
    
    valid_sizes = ["256x256", "512x512", "1024x1024", "1792x1024", "1024x1792"]
    if size not in valid_sizes:
        raise ValueError(f"size must be one of: {valid_sizes}")
    
    # Step 2: Get API key
    api_key = os.getenv("GROQ_API_KEY")
    if not api_key:
        raise ValueError("GROQ_API_KEY not set")
    
    # Step 3: Create Groq client
    client = Groq(api_key=api_key)
    
    # Step 4: Prepare parameters
    params = {
        "model": "dall-e-3",
        "prompt": prompt,
        "size": size,
        "quality": quality,
        "response_format": "b64_json"
    }
    
    if style:
        params["style"] = style
    
    # Step 5: Generate image
    response = client.images.generate(**params)
    
    # Step 6: Extract image data
    image_data = response.data[0]
    image_base64 = image_data.b64_json
    revised_prompt = getattr(image_data, 'revised_prompt', prompt)
    
    # Step 7: Return structured result
    return {
        "image_base64": image_base64,
        "size": size,
        "format": "png",
        "prompt_used": prompt,
        "revised_prompt": revised_prompt
    }
```

RÈGLES CRITIQUES IMAGE GENERATION:
- Valider le prompt (min 5 caractères)
- Utiliser des tailles standard prédéfinies
- Demander response_format="b64_json" pour avoir l'image en base64
- Model DOIT être "dall-e-3"
- Inclure le revised_prompt si disponible
