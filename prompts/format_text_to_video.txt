FORMAT EXACT POUR TEXT-TO-VIDEO - SUIT CETTE STRUCTURE:
```python
from groq import Groq
import os

def generate_video(prompt, duration=5, resolution="1024x576", fps=24):
    """
    Génère une vidéo depuis une description textuelle.
    
    Args:
        prompt: Description de la vidéo à générer (str)
        duration: Durée en secondes (int, défaut 5)
        resolution: Résolution "WIDTHxHEIGHT" (str, défaut "1024x576")
        fps: Images par seconde (int, défaut 24)
    
    Returns:
        dict avec clés:
            - video_url: URL de la vidéo générée
            - duration: Durée réelle de la vidéo
            - resolution: Résolution de la vidéo
            - format: Format du fichier (ex: 'mp4')
            - prompt_used: Prompt utilisé pour la génération
    """
    # Step 1: Validate input
    if not isinstance(prompt, str) or len(prompt) < 10:
        raise ValueError("prompt must be at least 10 characters")
    
    if not isinstance(duration, int) or duration < 1 or duration > 30:
        raise ValueError("duration must be between 1 and 30 seconds")
    
    # Step 2: Get API key
    api_key = os.getenv("GROQ_API_KEY")
    if not api_key:
        raise ValueError("GROQ_API_KEY not set")
    
    # Step 3: Create Groq client
    client = Groq(api_key=api_key)
    
    # Step 4: Generate video
    response = client.videos.generate(
        model="stable-video-diffusion",
        prompt=prompt,
        duration=duration,
        resolution=resolution,
        fps=fps
    )
    
    # Step 5: Extract video data
    video_url = response.url
    
    # Step 6: Return structured result
    return {
        "video_url": video_url,
        "duration": duration,
        "resolution": resolution,
        "format": "mp4",
        "prompt_used": prompt
    }
```

RÈGLES CRITIQUES TEXT-TO-VIDEO:
- Valider la longueur du prompt (min 10 chars)
- Limiter la durée (1-30 secondes typiquement)
- Utiliser des résolutions standard (1024x576, 1920x1080, etc.)
- Retourner l'URL ou path de la vidéo générée
